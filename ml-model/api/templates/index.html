📡 API Endpoints

GET /health

Health Check - Returns API status and model information

POST /predict

Crop Prediction - Predict optimal crop based on soil parameters

Request Body:

{
  "N": 90,
  "P": 45,
  "K": 50,
  "temperature": 25,
  "humidity": 70,
  "ph": 6.5,
  "rainfall": 120
}
POST /fertilizer-recommendation

Fertilizer Recommendation - Get personalized fertilizer recommendations

Request Body:

{
  "N": 50,
  "P": 30,
  "K": 40,
  "crop": "Rice",
  "soil_ph": 6.5,
  "area_hectares": 1.0
}
GET /model-info

Model Information - Get detailed information about ML models

GET /supported-crops

Supported Crops - List all supported crops

POST /batch-predict

Batch Prediction - Predict multiple samples at once
